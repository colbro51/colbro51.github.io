Camp Installer (Android)

To finish installing Camp:

Tap the ⋮ menu in Chrome (some devices use a ☰ menu)

Select Install App (or Add to Home Screen on some devices)

Then open Camp from the new icon.

Camp Installer (iPhone)

To finish installing Camp and unlock all features, including Maps, add it to your Home Screen.

  1. Tap the Share button
  2. Select Add to Home Screen
  3. Open Camp from the new icon

Tip: If you already installed Camp earlier, simply open it from your Home Screen. Safari can only show the installer.

Open Camp

You’ve already completed installation.

Open Camp from your Home Screen to use Maps and all features.

Safari can only show the installer and cannot run the full app.

Camp Installer (Windows)

Use your browser’s menu to install Camp.

Then open the Camp app from your Start menu or desktop.

Using Maps with the Camp App
It’s fine to open multiple Maps while exploring the app.

When you’re navigating for real, return to the Camp app before opening another route so only one Maps session runs at a time.

If you use the Camp app across multiple days, it helps to close any Maps sessions you opened earlier so they don’t build up in the background.

You can safely run other apps, like a GPS tracker, at the same time. The Camp app will resume normally whenever you bring it forward.

Google Maps Didn't Open

Your device didn't switch to the Google Maps app.

This can happen if:

  • The Maps app isn't installed
  • Your device opens links in the browser
  • iOS kept the link inside the PWA

You can still continue, but navigation may be limited.
